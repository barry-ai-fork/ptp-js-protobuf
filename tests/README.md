## Test

Run these tests with Node.js:

```bash
minipbjs> npm install --save-dev
minipbjs> cd tests/
tests   > node cli/test.js ...
```

or simply run all tests:

```bash
minipbjs> npm install -g tape
minipbjs> npm install --save-dev
minipbjs> npm run test
```
